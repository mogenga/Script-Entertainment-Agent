# 情感本DM演绎设计助手 - 实施计划

## 项目概述

**目标**：构建一个面向剧本杀DM的情感本演绎设计工具，通过多Agent协作系统，帮助DM设计一对一仪式动作式演绎环节。

**技术栈**：
- 前端：React Native + Expo
- 后端：FastAPI + SQLModel + PostgreSQL
- 大模型：Claude API（Prompt Engineering）

## 注意事项

1. **禁止emoji**：前端UI禁止使用emoji
2. **白色主题**：主色调为白色，按钮使用深色
3. **极简设计**：界面元素从简，功能优先
4. **不微调模型**：仅使用Prompt Engineering，不进行模型微调
5. **PostgreSQL统一**：开发生产统一使用PostgreSQL
