# 情感本DM演绎设计助手 - 设计规范

## 1. 设计原则

### 1.1 核心原则
- **极简主义**：界面元素从简，避免冗余装饰
- **功能优先**：所有设计服务于功能，不做无意义的视觉效果
- **清晰层级**：通过间距、字号、色彩建立明确的信息层级
- **专注沉浸**：减少干扰元素，让DM专注于剧本内容

### 1.2 禁用项
- **禁止使用emoji**作为UI图标或装饰
- 禁止使用复杂的动画效果
- 禁止使用渐变色背景（除非特殊情况）
- 禁止使用过多的阴影和边框

#### 列表

### 2.7 页面布局

#### 通用布局
- 页面边距：16px（移动端）
- 最大宽度：600px（内容区）
- 内容居中

#### 页面结构
```
[状态栏]

[内容区 - 可滚动]
```

### 2.8 动画规范

#### 允许的动画
- 页面切换：淡入淡出，200ms
- 按钮点击：透明度变化，100ms
- 列表加载：渐显，150ms

#### 禁止的动画
- 复杂的转场动画
- 持续的微动效
- 3D变换

### 3.5 Agent设计规范

#### Agent基类
```python
from abc import ABC, abstractmethod
from typing import Any, Dict

class BaseAgent(ABC):
    """Agent基类"""
    
    name: str = ""
    description: str = ""
    
    def __init__(self, llm_service: LLMService):
        self.llm = llm_service
    
    @abstractmethod
    async def execute(self, input_data: Dict[str, Any]) -> Dict[str, Any]:
        """执行Agent任务"""
        pass
    
    def _build_prompt(self, **kwargs) -> str:
        """构建Prompt（子类可覆盖）"""
        raise NotImplementedError
```

#### Prompt规范
- 使用结构化Prompt（System + User）
- 输出格式要求明确（JSON）
- 包含示例（few-shot）

### 3.6 错误处理

#### 自定义异常
```python
class AppException(Exception):
    """应用基础异常"""
    def __init__(self, code: int, error: str, message: str):
        self.code = code
        self.error = error
        self.message = message

class NotFoundError(AppException):
    def __init__(self, resource: str):
        super().__init__(404, "NOT_FOUND", f"{resource}不存在")
```

#### 全局异常处理
```python
@app.exception_handler(AppException)
async def app_exception_handler(request: Request, exc: AppException):
    return JSONResponse(
        status_code=exc.code,
        content={"code": exc.code, "error": exc.error, "message": exc.message}
    )
```
